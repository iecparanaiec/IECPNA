¿De qué se trata?

Una experiencia intensiva que distribuye el proceso completo de desarrollo en cinco jornadas, permitiendo aprender, aplicar, corregir y avanzar con mayor profundidad.

Una semana para construir una solución real

La WeekendClass toma la metodología FactorIA y la transforma en una experiencia de producción progresiva. Cada día se trabaja una etapa distinta: arquitectura, diseño, desarrollo, datos, automatización y publicación. El participante no observa una demostración: desarrolla su propio proyecto acompañado por el equipo IEC.

Metodología FactorIA

ChatGPT y asistentes de IA como arquitectos y copilotos.
AI Studio para diseño, prototipado y generación de soluciones.
Supabase para base de datos, autenticación y almacenamiento.
GitHub, automatizaciones y despliegue dentro de un flujo profesional.

Lo que vas a construir

El objetivo es finalizar la semana con una solución funcional, conectada y preparada para ser presentada o utilizada.

Página web profesional Responsive, moderna y lista para publicar.
Panel ERP Gestión de información y procesos internos.
Base de datos Tablas, registros, formularios y conexión real.
Automatizaciones Mensajes, correos, datos y procesos conectados.
IA integrada Asistentes, análisis y funciones inteligentes.
Proyecto versionado Repositorio, respaldo y estructura profesional.
Publicación online Dominio, despliegue y puesta en producción.
Modelo comercial Presentación, presupuesto y venta de la solución.

Cronograma de la semana

Cada jornada dura tres horas y tiene un objetivo concreto. El proyecto crece día tras día hasta llegar a producción.

Día 1 Lunes 14:00 a 17:00

Arquitectura, idea y planificación con IA

Definición del proyecto, análisis de requerimientos, usuarios, funciones, módulos y flujo general. Se utiliza IA para convertir una idea en una solución ordenada, viable y preparada para desarrollar.

Idea de negocio Requerimientos Arquitectura Prompts profesionales Plan de desarrollo
Resultado del día: proyecto definido, estructura funcional y plan de trabajo completo.
Día 2 Martes 14:00 a 17:00

Diseño de la web y del panel ERP

Construcción de la identidad visual, página pública, navegación, formularios y panel administrativo. Se trabaja sobre interfaces responsive y experiencia de usuario con asistencia de IA.

AI Studio Frontend UX/UI Landing Dashboard ERP
Resultado del día: frontend funcional con página web y panel de gestión.
Día 3 Miércoles 14:00 a 17:00

Base de datos, formularios y backend

Modelado de información, creación de tablas, relaciones y registros. Conexión de formularios, autenticación y operaciones básicas para que el sistema almacene y gestione datos reales.

Supabase SQL Tablas Autenticación CRUD
Resultado del día: solución conectada a una base de datos en la nube.
Día 4 Jueves 14:00 a 17:00

Automatizaciones, integraciones e IA aplicada

Diseño de procesos automáticos para conectar formularios, correos, mensajes, bases de datos y asistentes. Se incorporan webhooks, disparadores y funciones inteligentes dentro del flujo operativo.

Automatizaciones Webhooks Correos Mensajería IA aplicada
Resultado del día: procesos empresariales conectados y funcionando de forma automática.
Día 5 Viernes 14:00 a 17:00

GitHub, publicación, presentación y venta

Organización final del proyecto, control de versiones, despliegue online, pruebas, correcciones y preparación comercial. La jornada cierra con la presentación de cada solución y un plan para ofrecerla a clientes.

GitHub Deploy Dominio Pruebas Venta digital
Resultado del día: proyecto publicado, presentado y listo para ser utilizado o comercializado.
