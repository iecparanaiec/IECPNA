LA DIFERENCIA: GENERAR Y EDITAR

🖼️

Imagen estática

Una IA genera el flyer como una sola imagen.

• El texto queda integrado.

• Mover objetos es difícil.

• Corregir una palabra puede requerir regenerar.

• La composición está cerrada.

🧩

Diseño editable

Canva trabaja con objetos que se pueden intervenir.

• Cambiar textos.

• Mover imágenes.

• Ajustar colores.

• Duplicar, borrar o reemplazar elementos.

“Creá una imagen publicitaria para ________, dirigida a ________. Debe transmitir ________, utilizar una estética ________, una paleta de ________ y una composición limpia y profesional.”

Actividad 4

CREAR UNA PRESENTACIÓN

Estructura sugerida

1. Portada.

2. Problema o contexto.

3. Idea principal.

4. Beneficios.

5. Ejemplos.

6. Llamado a la acción.

Prompt sugerido

“Creá una presentación de ocho diapositivas sobre ________, dirigida a ________. Usá un tono ________, una estética ________, textos breves y una diapositiva final con llamada a la acción.”

REVISIÓN DE LA PRESENTACIÓN

Contenido

• Una idea principal por diapositiva.

• Textos breves y claros.

• Secuencia lógica.

• Ejemplos comprensibles.

Diseño

• Buena jerarquía visual.

• Contraste suficiente.

• Identidad coherente.

• Imágenes útiles, no decorativas.

COPIAR EL ENLACE PÚBLICO

🔗

Una vez publicado, el formulario obtiene un enlace que cualquier persona puede abrir desde su celular o computadora.

1. Copiar

Guardá la URL publicada.

2. Probar

Abrila en otro dispositivo.

3. Compartir

Usala en QR, redes o mensajes.

GENERAR EL CÓDIGO QR

Representación visual de ejemplo.

Paso a paso

1. Abrí la sección de aplicaciones.

2. Buscá “Generador de código QR”.

3. Pegá el enlace del formulario.

4. Generá el código.

5. Insertalo dentro del diseño.

CREAR EL FLYER FINAL CON QR

El flyer debe incluir:

• Título claro.

• Beneficio principal.

• Instrucción para escanear.

• Código QR visible.

• Alternativa de contacto.

• Identidad visual coherente.

Prompt sugerido

“Diseñá un flyer para invitar a las personas a completar un formulario. Incluí el texto ________, una zona destacada para colocar un QR y la instrucción ‘Escaneá para participar’. Usá la identidad visual ________.”
