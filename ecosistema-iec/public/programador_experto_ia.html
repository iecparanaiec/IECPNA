Fundamentación

La programación se transforma de manera constante. Por eso, la formación no puede terminar en un programa cerrado: necesita continuidad, práctica y actualización permanente.

Aprender a programar en la era de la IA

El desarrollador actual utiliza Inteligencia Artificial para analizar requerimientos, diseñar arquitecturas, generar código, detectar errores, documentar, crear pruebas y acelerar entregas. El taller enseña programación real incorporando la IA como prompter, copiloto y herramienta de trabajo desde el comienzo.

Formación continua dentro de un entorno productivo

Ingreso progresivo: cada alumno avanza según su nivel y experiencia.
Actualización constante ante nuevas herramientas, lenguajes y metodologías.
Desarrollo de soluciones funcionales y participación en proyectos reales.
Evaluación permanente para acceder a oportunidades dentro de FactorIA.

No es un curso que termina: es un taller que evoluciona

Los participantes pueden continuar asistiendo para profundizar tecnologías, actualizar conocimientos, construir portfolio, integrarse a equipos de trabajo y resolver proyectos de complejidad creciente. Los contenidos se renuevan según la evolución del sector y las necesidades de los proyectos de FactorIA.

Objetivos generales

Formar desarrolladores capaces de crear, integrar, automatizar y mantener soluciones digitales en contextos reales.

Desarrollar bases sólidas

Comprender lógica, estructuras, funciones, objetos, datos, algoritmos y buenas prácticas.

Programar junto a la IA

Usar IA para planificar, generar, depurar, probar, documentar y mejorar código.

Construir soluciones completas

Crear frontend, backend, bases de datos, APIs, autenticación y paneles de gestión.

Desarrollar experiencia profesional

Trabajar con requerimientos, equipos, entregas y proyectos vinculados al ecosistema FactorIA.

Recorrido formativo permanente

Los módulos funcionan como etapas de crecimiento. El alumno puede profundizar, repetir prácticas y continuar con nuevos proyectos.

I

Programación moderna con Inteligencia Artificial

Introducción a la lógica, resolución de problemas y uso de IA como prompter, copiloto, depurador y asistente técnico. Se aprende a solicitar código, validarlo, comprenderlo y mejorarlo.

Lógica Algoritmos Prompts técnicos Debug
II

Desarrollo web con HTML y CSS

Construcción de landing pages, sitios institucionales e interfaces responsive. Desde esta etapa, los participantes pueden comenzar a desarrollar piezas supervisadas para portfolio y FactorIA.

HTML CSS Responsive UX/UI
III

JavaScript y aplicaciones interactivas

Variables, funciones, objetos, DOM, eventos, validación, almacenamiento, asincronía y consumo de APIs. Desarrollo de aplicaciones que responden a acciones reales del usuario.

JavaScript DOM Eventos APIs
IV

Frontend moderno con React

Componentes, estados, hooks, rutas, formularios y paneles administrativos. Construcción de aplicaciones escalables conectadas con servicios y APIs.

React Componentes Hooks Dashboards
V

Backend con Node y Express

Servidores, rutas, controladores, APIs REST, autenticación, validaciones, archivos y conexión entre frontend, servicios y bases de datos.

Node.js Express REST API Autenticación
VI

Bases de datos SQL y NoSQL

Modelado, tablas, relaciones, consultas y seguridad. Trabajo con Supabase, PostgreSQL, MySQL y MongoDB según las necesidades de cada proyecto.

SQL Supabase PostgreSQL MongoDB
VII

Python y herramientas internas

Scripts, lectura de archivos, procesamiento de datos, utilidades administrativas y automatización de tareas que complementan aplicaciones y sistemas.

Python Scripts Datos Utilidades
VIII

Automatizaciones e integraciones

Creación de flujos con n8n, Make y Zapier. Webhooks, APIs, formularios, correos, mensajería, calendarios, documentos, OCR y sincronización de datos.

n8n Make Zapier Webhooks
IX

Inteligencia Artificial aplicada al software

Construcción de copilotos, agentes, asistentes especializados, bases de conocimiento y funciones inteligentes integradas dentro de aplicaciones y procesos.

AI Studio Agentes RAG Copilotos
X

C++ y Arduino para hardware e IoT

Esta etapa avanzada introduce C++, Arduino IDE, sensores, motores, comunicación serial e integración entre dispositivos físicos, aplicaciones web y automatizaciones.

C++ Arduino Sensores IoT
XI

Git, despliegue y producción

Versionado, repositorios, servidores, hosting, dominios, certificados SSL, variables de entorno, mantenimiento y publicación de aplicaciones reales.

GitHub Deploy Servidor Producción
XII

Proyectos permanentes y especialización

Luego de adquirir las bases, el alumno continúa trabajando en proyectos de mayor complejidad, nuevas tecnologías y áreas de especialización según su perfil y los desarrollos activos.

Portfolio Trabajo en equipo Especialización Actualización
